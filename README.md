#  To-Do List Website

A simple and interactive **To-Do List web application** built using **HTML, CSS, and JavaScript**. This project helps users add, manage, and organize daily tasks in a clean and user-friendly interface. It is ideal for beginners learning front-end web development.

##  Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Responsive and simple UI
* Runs directly in the browser (no backend required)

##  Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Functionality and interactivity

### Prerequisites

You only need a modern web browser such as:

* Google Chrome
* Mozilla Firefox
* Microsoft Edge

### Run the Project

1. Open the project folder
2. Double-click on `index.html`
3. The To-Do website will open in your browser

##  How It Works

1. User enters a task in the input field
2. Clicks the **Add** button (or presses Enter)
3. The task appears in the list
4. Tasks can be marked as completed or removed
5. JavaScript dynamically updates the task list
